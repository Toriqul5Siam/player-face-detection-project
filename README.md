# Player Face Detection Project

This project uses a YOLOv8 deep learning model to detect people (players) in a football video.

## Objective
To demonstrate computer vision techniques for detecting human presence in video data.

## Dataset
A real-world football goal celebration video sourced from Pexels, renamed as "video 1.mp4".

## Dataset Note
The video file is not included in this repository due to GitHub file size limits.

It is included in the submitted ZIP file for the assignment.

## Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV

## Requirements
Python 3.8+
pip install -r requirements.txt

## Run
python player_face_detection.py

## Output
Results will be saved in the "results" folder.

## Note
YOLOv8 model weights (yolov8n.pt) will be downloaded automatically on first run.
